# V2 Subagent Prompt Logs

These files record the subagent prompts used during the V2 idiomatic evolution experiment on 2026-05-21.

They are included for reproducibility: each phase file captures the prompt text sent to the isolated worker for that phase. Phase 1 includes both the original worker prompt that disconnected and the replacement worker prompt that completed the phase, because both were part of the actual session history.

Branch used during the experiment: `experiment/v2-idiomatic-evolution`.
